## ⚡ Technical Setup

1. **Clone & Install:**
   ```bash
   git clone https://github.com/Prolayjit-B14/prolayjit-biswas.git
   npm install
   ```

2. **Environment Configuration:**
   Create a `.env.local` with:
   * `GITHUB_TOKEN` (Meta-data fetching)
   * `RESEND_API_KEY` (Contact infrastructure)

3. **Development:**
   ```bash
   npm run dev
   ```
